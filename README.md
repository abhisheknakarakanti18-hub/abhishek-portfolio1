# Personal Portfolio Website (GitHub Pages)

This is a simple one-page portfolio you can host for free using GitHub Pages.

## Files
- `index.html` — your website
- `resume.pdf` — your resume (add this file)

## Quick Start (Local)
Open `index.html` in a browser.

## Publish on GitHub Pages
1. Create a GitHub repo (example name: `abhishek-portfolio`)
2. Upload `index.html` and `resume.pdf`
3. Go to **Settings → Pages**
4. Under **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Your site will be live at:
   `https://YOUR_GITHUB_USERNAME.github.io/abhishek-portfolio/`

## Customize
- Replace `YOUR_EMAIL_HERE` in `index.html`
- Update Projects section with links and bullet points
